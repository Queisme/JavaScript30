  1. map() - creates a new array with the results of calling a provided function on every element in this array
    ex:
    const arr = [1, 2, 3, 4, 5, 6];
    const mapped = arr.map(element => element + 30);
    console.log(mapped); // [31, 32, 33, 34, 35, 36]

  2. filter() - creates a new array with only elements that passes the condition inside the provided function
    ex:
    const arr = [1, 2, 3, 4, 5, 6];
    const filtered = arr.filter(element => element === 2 || element === 4);
    console.log(filtered); // [2, 4]

  3. sort() - used to arrange/sort array‚Äôs elements either in ascending or descending order
    ex:
    const arr = [1, 2, 3, 4, 5, 6];
    const alpha = ['f', 'a', 'c', 'v', 'z'];

    //sort in descending order
    descend = arr.sort((a, b) => a > b ? -1 : 1); console.log(desend); // [6, 5, 4, 3, 2, 1]

    //sort in ascending order
    ascend = alpha.sort((a, b) => a > b ? 1 : -1); console.log(ascend);// ['a', 'c', 'f', 'v', 'z']

  4. forEach() - loops over array by executing a provided callback function for each element in an array
    ex:
    const arr = [1, 2, 3];
    arr.forEach(element => {console.log(element);});
    // 1
    // 2
    // 3

  5. concat() - merges two or more arrays and returns a new array, without changing the existing arrays
    ex:
    const arr1 = ['a', 'b', 'c'];
    const arr2 = ['d', 'e', 'f'];

    console.log(arr1.concat(arr2)); // ['a', 'b', 'c', 'd', 'e', 'f']
    console.log(arr1); // ['a', 'b', 'c']
    console.log(arr2); // ['d', 'e', 'f']

  6. every() - checks every element in the array that passes the condition, returning true or false as appropriate
    ex:
    const arr = [1, 2, 3, 4, 5, 6, 7];

    //all elements are greater than 5
    const greaterFive = arr.every(num => num > 5);
    console.log(greaterFive); // false

    //all elements are less than 9
    const lessNine = arr.every(num => num < 9);
    console.log(lessNine); // true

  7. some() - checks if at least one element in the array that passes the condition, returning true or false as appropriate

  8. includes() - checks if an array includes the element that passes the condition, returning true or false as appropriate

  9. join() - returns a new string by concatenating all of the array‚Äôs elements separated by the specified separator

  10. reduce() - applies a function against an accumulator and each element in the array to reduce it to a single value

  11. find() - returns the value of the first element in an array that pass the test in a testing function

  12. findIndex() - returns the index of the first element in an array that pass the test in a testing function

  13. indexOf() - returns the index of the first occurrence of the specified element in the array, or -1 if it is not found

  14. fill() - fills the elements in an array with a static value and returns the modified array

  15. slice() - returns a new array with specified start to end elements

  16. reverse() - reverses an array in place. Element at last index will be first and element at 0 index will be last

  17. push() - adds one or more elements to the end of array and returns the new length of the array

  18. pop() - removes the last element from the end of array and returns that element

  19. shift() - removes the first element from an array and returns that element

  20. unshift() - adds one or more elements to the beginning of an array and returns the new length of the array
